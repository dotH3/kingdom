# Welcome to Kingdom

This is the first post on Kingdom, a minimalist blog built with Astro.

## What is Kingdom?

Kingdom is a static blog that focuses on simplicity and readability. No unnecessary JavaScript, no complex frameworks - just clean, fast content.

## Features

- **Minimalist design** with an aesthetic brown color scheme
- **Dark mode support** that respects your system preferences
- **Fast loading** thanks to static site generation
- **SEO optimized** for better discoverability

## Writing Posts

Posts are written in Markdown and include frontmatter with the following fields:

- `title`: The post title
- `date`: Publication date
- `desc`: Short description
- `tags`: Array of tags
- `slug`: URL-friendly identifier
- `draft`: Whether the post is a draft

Enjoy writing!
